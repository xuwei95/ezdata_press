## 表格示例

| 参数         | 说明               | 类型            | 可选值                                           | 默认值  |
| ------------ | ------------------ | --------------- | ------------------------------------------------ | ------- |
| `type`       | 类型               | string          | `default` `primary` `success` `danger` `warning` | default |
| `font-size`  | 副标题文字大小     | string / number | ——                                               | 15px    |
| `title-size` | 主标题文字大小     | string / number | ——                                               | 17px    |
| `bold`       | 文字是否以粗体显示 | boolean         | ——                                               | false   |
| `center`     | 是否居中           | boolean         | ——                                               | false   |
